# AI助手智能体管理系统

## 功能概述

这是一个智能体管理系统，包含以下主要功能：

1. **AI助手主页面** (`/ai-assistant`) - 显示所有智能体卡片
2. **智能体编辑页面** (`/agent-editor`) - 用于创建和编辑智能体

## 使用方法

### 1. 访问AI助手页面

- 在导航栏中点击"AI助手"
- 或者直接访问 `/ai-assistant` 路由

### 2. 创建新智能体

- 在AI助手页面顶部点击"创建智能体"按钮
- 系统会跳转到智能体编辑页面，进入创建模式

### 3. 编辑现有智能体

- 在任意智能体卡片上点击"编辑"按钮
- 系统会跳转到智能体编辑页面，进入编辑模式
- 会传递智能体的ID和名称等参数

### 4. 智能体编辑页面功能

#### 左侧编排面板
- **提示词编辑器**: 编写AI智能体的核心提示词
- **变量管理**: 定义输入变量（如：店铺名称、目标受众等）
- **知识库配置**: 上传和管理相关文档
- **元数据过滤**: 设置数据过滤规则
- **工具配置**: 配置可用的工具和功能
- **视觉设置**: 控制视觉相关功能

#### 右侧调试预览面板
- **Agent设置**: 选择AI模型和配置参数
- **输入变量调试**: 测试不同输入值的效果
- **实时预览**: 查看AI生成的回复内容

#### 底部聊天界面
- **聊天测试**: 与智能体进行对话测试
- **功能状态**: 显示当前功能开启状态
- **管理入口**: 快速访问管理功能

## 技术特性

- **Vue 3**: 使用最新的Vue.js框架
- **Vue Router**: 单页面应用路由管理
- **响应式设计**: 支持不同屏幕尺寸
- **组件化架构**: 模块化的代码结构

## 路由配置

```javascript
// 主要路由
{ path: '/ai-assistant', component: AIAssistant, meta: { hideFooter: true } }
{ path: '/agent-editor', component: AgentEditor, meta: { hideFooter: true } }
```

## 数据流

1. 用户点击"创建智能体" → 跳转到编辑页面（创建模式）
2. 用户点击"编辑" → 跳转到编辑页面（编辑模式，传递智能体ID）
3. 编辑页面根据路由参数判断模式，加载相应数据
4. 保存/发布后返回列表页面

## 开发说明

### 文件结构
```
src/
├── views/
│   ├── AIAssistant.vue      # AI助手主页面
│   └── AgentEditor.vue      # 智能体编辑页面
├── components/
│   └── MainNavbar.vue       # 导航栏组件
└── main.js                  # 路由配置
```

### 添加新功能
1. 在相应的Vue组件中添加新的方法
2. 更新路由配置（如需要）
3. 添加相应的样式和交互逻辑

## 注意事项

- 所有智能体数据目前使用模拟数据，实际项目中需要连接后端API
- 编辑页面的返回按钮使用浏览器历史记录，确保用户体验
- 响应式设计确保在不同设备上的良好显示效果
