# 教学管理工作量化页面

## 功能特性

### 1. 筛选功能
- **按课程筛选**：支持选择特定课程进行数据筛选
- **按类型筛选**：支持按理论课、实践课、实验课进行筛选
- **应用筛选**：点击应用筛选按钮执行筛选操作
- **重置筛选**：一键清空所有筛选条件

### 2. 数据可视化
- **饼图展示**：课程类型工作量占比（理论课、实践课、实验课）
- **柱状图展示**：近5年各学期工作量变化趋势
- **响应式布局**：饼图和柱状图按1:2比例排列，支持移动端适配

### 3. 课表日历
- **月份切换**：支持查看不同月份的课表
- **课程显示**：显示具体课程时间、名称和类型
- **类型标识**：不同课程类型用不同颜色标识
- **交互功能**：点击日期可查看详细信息

## 技术实现

### 前端框架
- Vue 3 + Composition API
- Element Plus UI组件库
- ECharts图表库

### 主要功能模块
1. **筛选组件**：基于Element Plus的Select组件
2. **图表组件**：使用ECharts实现饼图和柱状图
3. **日历组件**：自定义实现的月历组件
4. **响应式设计**：支持桌面端和移动端

### 数据模拟
- 课程数据：模拟了多种课程类型和名称
- 工作量数据：模拟了近5年的工作量变化
- 课表数据：模拟了本月的课程安排

## 使用方法

1. 启动开发服务器：`npm run dev`
2. 访问页面：`http://localhost:5173/#/teaching-resources/workload`
3. 使用筛选功能查看不同维度的数据
4. 通过图表了解工作量分布和趋势
5. 查看日历了解课程安排

## 页面结构

```
工作量化页面
├── 页面标题
├── 筛选区域
│   ├── 课程筛选
│   ├── 类型筛选
│   └── 操作按钮
├── 图表区域
│   ├── 饼图（1/3宽度）
│   └── 柱状图（2/3宽度）

```
