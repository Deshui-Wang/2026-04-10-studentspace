# 学生作业管理功能

## 功能概述
在教学管理二级菜单中的"教学经验"下面新增了"学生作业"菜单，提供全面的学生作业管理功能。

## 功能特性

### 1. 筛选功能
- **班级筛选**：按班级筛选作业数据
- **作业名称搜索**：支持按作业名称进行搜索
- **学生姓名搜索**：支持按学生姓名进行搜索

### 2. 统计概览
- 总作业数统计
- 平均提交率显示
- 平均正确率显示
- 参与学生数统计

### 3. 提交率统计
- **已提交学生**：显示已提交作业的学生信息（头像、姓名、班级、提交时间）
- **未提交学生**：显示未提交作业的学生信息（头像、姓名、班级）

### 4. 正确率统计
- **优秀学生**：显示90分以上的学生（头像、姓名、班级、分数）
- **需要关注的学生**：显示60分以下的学生（头像、姓名、班级、分数）

### 5. 知识点分布
- **简单题目**：显示简单题目的数量和占比
- **正常题目**：显示正常难度题目的数量和占比
- **较难题目**：显示较难题目的数量和占比
- 使用进度条可视化显示各难度级别的分布

### 样式设计
- 现代化的卡片式布局

## 使用方法
1. 进入"教学管理"模块
2. 点击左侧菜单中的"学生作业"
3. 使用筛选功能查找特定数据
4. 查看各项统计信息和学生详情
