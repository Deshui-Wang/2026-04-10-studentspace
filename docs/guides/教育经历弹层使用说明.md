# 教育经历弹层功能说明

## 功能概述
为教育经历页面添加了"添加教育经历"弹层功能，包含以下特性：

### 1. 文件上传功能
- 支持拖拽上传和点击上传
- 支持的文件格式：jpg、jpeg、png、PDF
- 显示文件预览和文件大小
- 支持删除已选择的文件

### 2. 学校选择功能
- 内置全国高校名称数据库（包含200+所高校）
- 支持联想输入和实时搜索
- 下拉列表显示匹配的学校名称
- 点击选择学校

### 3. 表单录入功能
- 专业名称：文本输入框
- 学历选择：下拉选择（专科、本科、硕士、博士、博士后）
- 时间选择：入学时间和毕业时间（日期选择器）

### 4. 表单验证
- 所有必填字段都有验证
- 只有所有必填字段都填写完整才能提交
- 提交按钮在表单不完整时会被禁用

## 文件结构
```
src/components/profile/
├── Education.vue              # 主页面，包含弹层引用
├── AddEducationModal.vue      # 弹层组件
└── Education.vue.backup       # 原文件备份
```

## 使用方法
1. 在 `Education.vue` 页面点击"添加教育经历"按钮
2. 弹层会打开，显示表单界面
3. 填写所有必填信息
4. 点击"确认添加"提交表单
5. 点击"取消"或点击弹层外部区域关闭弹层

## 技术特性
- 使用 Vue 3 Composition API
- 响应式设计，支持移动端
- 现代化的 UI 设计
- 完整的表单验证
- 文件上传预览功能
- 学校名称自动补全

## 自定义配置
如需修改学校列表，请编辑 `AddEducationModal.vue` 中的 `universities` 数组。
如需修改文件上传限制，请修改 `accept` 属性和文件类型检查逻辑。
