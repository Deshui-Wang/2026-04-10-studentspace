# 报告数据二级菜单修改说明

## 修改内容

根据用户要求，已成功从报告数据页面中删除以下三个二级菜单项，仅保留"档案报告"：

### 删除的菜单项
- **教学报告** (teaching)
- **绩效报告** (performance)
- **数据分析** (analysis)
- **导出管理** (export)

### 保留的菜单项
- **档案报告** (archive)

## 修改详情

### 1. 菜单栏简化
- 删除了4个不需要的tab-item元素
- 只保留"档案报告"一个菜单项
- 设置该菜单项为默认激活状态

### 2. 内容区域简化
- 删除了所有条件渲染的placeholder-content区域
- 只保留一个固定的档案报告内容区域
- 移除了不必要的条件判断逻辑

### 3. 代码优化
- 简化了模板结构
- 更新了默认的activeSubTab值为'archive'
- 保持了代码的整洁性和可维护性
