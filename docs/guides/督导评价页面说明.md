# 督导评价页面功能说明

## 页面概述
督导评价页面是一个完整的教学评价管理系统，用于展示和管理督导对教师教学的评价数据。

## 主要功能

### 1. 筛选区域
- **班级筛选**：支持按班级筛选评价数据
- **课程筛选**：支持按课程筛选评价数据  
- **搜索功能**：支持按课程名称或班级名称进行搜索

### 2. 词云标签
- 显示高频评价词汇
- 包含：教学认真、内容充实、方法得当、互动良好、逻辑清晰、重点突出、启发思维、技术运用等
- 不同词汇有不同的颜色和大小，体现使用频率

### 3. 数据列表
包含以下字段：
- **课程名称**：被评价的课程
- **班级名称**：被评价的班级
- **听课类型**：随机听课 / 教学检查
- **教学态度**：评分（0-10分）+ 进度条显示
- **备课情况**：评分（0-10分）+ 进度条显示
- **教学熟练度**：评分（0-10分）+ 进度条显示
- **教学纪律**：评分（0-10分）+ 进度条显示
- **教学内容**：标签形式显示（符合大纲、重点突出、信息量适中、反映前沿）
- **教学方法**：标签形式显示（启发互动、逻辑清晰、语言表达、技术运用）
- **课堂情况**：学生参与度百分比、课堂纪律状态、学习效果状态
- **教学效果**：目标达成百分比、学生理解掌握百分比
- **针对性指导**：文字内容预览
- **操作**：查看详情按钮

### 4. 详情弹窗
点击"查看详情"按钮可查看完整的督导评价报告，包括：
- 基本信息（课程名称、班级名称、听课类型、评价时间）
- 评分详情（各项评分的详细展示）
- 教学内容和方法（标签形式展示）
- 课堂情况（详细数据）
- 教学效果（详细数据）
- 针对性指导（完整文字内容）

## 技术特点

### 响应式设计
- 支持桌面端和移动端
- 表格在小屏幕上可横向滚动
- 筛选区域在移动端自动调整布局

### 交互体验
- 悬停效果和过渡动画
- 颜色编码的评分系统
- 直观的进度条显示
- 模态弹窗展示详细信息

### 数据可视化
- 词云标签展示高频词汇
- 进度条直观显示评分
- 颜色编码区分不同状态
- 标签系统展示多选属性

## 使用方法

1. 访问数据中心页面
2. 点击"评价数据"卡片
3. 在二级菜单中选择"督导评价"
4. 使用筛选条件查找特定评价
5. 点击"查看详情"查看完整评价报告
