# 学生评教页面功能说明

## 概述
新增的"学生评教"页面提供了完整的学生评价数据管理和查看功能，包含筛选、词云展示、数据列表和详细评价查看等功能。

## 功能特性

### 1. 筛选区域
- **班级筛选**：支持按班级筛选评价数据
- **课程筛选**：支持按课程筛选评价数据  
- **搜索功能**：支持按课程名称或班级名称进行模糊搜索

### 2. 词云标签
- 展示高频评价词汇
- 不同颜色和大小表示词汇使用频率
- 支持鼠标悬停效果

### 3. 数据列表
- 显示课程名称、班级名称、学生总数、好评率
- 好评率以进度条形式直观展示
- 支持点击查看详细评价

### 4. 评价详情弹窗
- 显示40-46人的虚拟评价数据
- 包含四个评分维度：
  - 教学态度（80-100分）
  - 教学内容（80-100分）
  - 教学方法（80-100分）
  - 学习效果（80-100分）
- 包含文字描述：
  - 优点描述
  - 改进建议

## 使用方法

1. 在评价数据页面点击"学生评教"标签
2. 使用筛选条件缩小数据范围
3. 查看词云了解高频评价词汇
4. 点击"查看详情"按钮查看具体评价
5. 在弹窗中浏览所有学生的详细评价

## 数据说明

当前使用虚拟数据进行展示，包含：
- 5个课程的评价数据
- 每个课程40-46个学生的评价
- 随机生成的评分和文字描述
- 真实的高频评价词汇
