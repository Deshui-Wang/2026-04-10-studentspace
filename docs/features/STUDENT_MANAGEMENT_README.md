# 学生管理页面

## 功能概述

在教学数据的二级菜单中，我在"班课"和"作业"之间添加了一个"学生"标签页，创建了完整的学生管理功能。

## 主要功能

### 1. 筛选区域
- **搜索框**：支持按学号或姓名进行搜索
- **实时筛选**：输入即时过滤学生列表

### 2. 数据列表
完整的学生信息表格，包含以下字段：

#### 基本信息
- **学号**：学生唯一标识，固定在左侧
- **姓名**：学生姓名，固定在左侧
- **班级**：所属班级

#### 学习表现
- **出勤率**：显示百分比，用不同颜色标签表示等级
- **课堂互动**：5星评分制，显示具体分数
- **作业完成率**：进度条显示，百分比形式
- **作业正确率**：进度条显示，根据正确率显示不同颜色
- **平均成绩**：数字显示，不同分数段用不同颜色标识
- **GPA**：标签形式显示，不同等级用不同颜色

#### 个性化信息
- **MBTI**：学生性格类型标签
- **兴趣画像**：多个标签显示学生兴趣爱好
- **科研项目**：显示项目名称，无项目显示"暂无"
- **专业匹配度**：圆形进度条显示匹配程度
- **就业建议**：详细的职业发展建议，固定在右侧

### 3. 交互功能
- **分页显示**：支持10/20/50/100条记录切换
- **表格排序**：支持各列排序（Element Plus表格默认功能）
- **响应式设计**：适配不同屏幕尺寸

## 技术特性

### 数据可视化
- **进度条**：用于完成率、正确率等百分比数据
- **圆形进度条**：用于专业匹配度
- **评分星级**：用于课堂互动评分
- **颜色标签**：用于出勤率、GPA等等级显示
- **多彩标签**：用于兴趣画像展示

### 响应式表格
- **固定列**：学号和姓名固定在左侧，就业建议固定在右侧
- **最大高度**：表格最大高度600px，超出显示滚动条
- **加载状态**：支持loading效果
- **边框样式**：条纹和边框增强可读性

### 智能筛选
- **实时搜索**：支持学号和姓名模糊匹配
- **大小写不敏感**：搜索不区分大小写
- **自动重置分页**：搜索时自动回到第一页

## 样本数据

页面包含8个样本学生数据，展示了不同类型的学生特征：

1. **优秀学生**：高出勤率、高GPA、有科研项目
2. **普通学生**：中等表现、不同专业匹配度
3. **需要关注的学生**：较低出勤率、需要额外指导

每个学生都有完整的档案信息，包括MBTI性格、兴趣爱好、科研项目（部分学生）和个性化就业建议。

## 颜色方案

### 出勤率标签
- 绿色（success）：95%以上
- 橙色（warning）：85-94%
- 红色（danger）：85%以下

### GPA标签
- 绿色（success）：3.7以上
- 橙色（warning）：3.0-3.6
- 红色（danger）：3.0以下

### 成绩颜色
- 绿色（优秀）：90分以上
- 蓝色（良好）：80-89分
- 橙色（中等）：70-79分
- 红色（较差）：70分以下

## 使用说明

1. 在数据中心页面点击"教学"卡片
2. 在教学数据的二级菜单中点击"学生"标签
3. 使用搜索框按学号或姓名筛选学生
4. 查看完整的学生信息表格
5. 使用分页功能浏览更多学生数据

这个学生管理页面为教师提供了全面的学生信息视图，帮助教师更好地了解每个学生的学习状况、性格特点和发展潜力，从而提供更个性化的教学指导。
