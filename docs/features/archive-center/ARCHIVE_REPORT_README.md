# 档案报告页面

## 功能概述

这是一个全新的档案报告页面，专为数据中心模块设计。该页面提供了完整的报告管理功能，包括筛选、搜索、创建、预览和下载等功能。

## 主要功能

### 1. 筛选区域
- **时间范围筛选**：支持今天、本周、本月、本季度、本年等时间范围筛选
- **搜索功能**：支持按报告名称和描述进行关键词搜索
- **创建按钮**：快速创建新报告

### 2. 数据展示
- **报告卡片**：以卡片形式展示报告信息
- **缩略图**：支持报告缩略图显示
- **格式标签**：显示报告格式（Word、PDF、PPT等）
- **详细信息**：显示文件大小、创建时间等信息

### 3. 交互功能
- **预览功能**：点击预览按钮查看报告详情
- **下载功能**：支持报告下载
- **创建功能**：集成报告创建模态框

## 技术特性

### 响应式设计
- 桌面端：网格布局，自适应列数
- 平板端：2列布局
- 移动端：单列布局，优化触摸交互

### 现代化UI
- 卡片式设计
- 渐变背景
- 平滑动画
- 悬停效果

### 组件化架构
- 复用个人中心的ReportCreationModal组件
- 清晰的组件通信
- 模块化的样式设计

## 使用方法

1. 在数据中心页面点击"报告"卡片
2. 进入档案报告页面
3. 使用筛选和搜索功能查找报告
4. 点击"创建"按钮创建新报告
5. 点击"预览"或"下载"按钮操作报告

## 样式特点

- 使用现代化的卡片设计
- 支持多种报告格式的图标显示
- 响应式布局适配不同屏幕尺寸
- 统一的色彩方案和交互效果
