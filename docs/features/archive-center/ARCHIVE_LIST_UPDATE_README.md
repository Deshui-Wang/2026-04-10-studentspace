# 档案列表页面新增更新

## 修改概述

已成功创建新的"档案列表"页面组件 `ArchiveList.vue`，用于"数据中心-报告数据-档案报告"调用使用。该页面提供了完整的档案管理功能，包括筛选、搜索、列表展示、操作等功能。

## 主要修改内容

### 1. 新增档案列表组件

#### ArchiveList.vue 新建
**位置**：`src/components/data-center/ArchiveList.vue`

**功能特性**：
- **筛选区域**：时间范围筛选、搜索框
- **档案列表**：表格形式展示档案信息
- **操作功能**：预览、下载、编辑、删除
- **分页功能**：支持分页浏览
- **响应式设计**：适配移动端和桌面端

### 2. 档案列表字段设计

按照用户要求，档案列表包含以下字段：

#### 表格列定义
1. **档案名称** - 显示档案标题和描述
2. **档案格式** - PDF、Word、PPT、压缩包等格式标签
3. **用途** - 晋升、评优、其他用途分类
4. **资料附件** - 附件数量和总大小
5. **创建时间** - 创建时间和更新时间
6. **已阅人次** - 查看次数统计
7. **操作** - 预览、下载、编辑、删除按钮

#### 数据字段结构
```javascript
{
  id: 1,
  name: '档案名称',
  description: '档案描述',
  format: 'pdf',           // 文件格式
  formatText: 'PDF',       // 格式显示文本
  purpose: 'promotion',    // 用途类型
  purposeText: '晋升',     // 用途显示文本
  attachmentCount: 3,      // 附件数量
  totalSize: 2048576,     // 总文件大小（字节）
  createTime: '2024-01-15 10:30:00',  // 创建时间
  updateTime: '2024-01-20 14:25:00',  // 更新时间
  viewCount: 15            // 查看次数
}
```
