# 课堂互动页面功能说明

## 功能概述
在数据中心-教学数据模块中新增了"课堂互动"二级菜单页面，用于展示和分析课堂互动数据。

## 实现内容

### 1. 页面组件
- 创建了 `src/components/teaching-resources/ClassroomInteraction.vue` 组件
- 页面包含筛选区域和数据列表两个主要部分

### 2. 筛选区域
参考出勤页面的筛选区域设计，包含以下筛选条件：
- **课程**：下拉选择框，支持全部课程筛选
- **班级**：下拉选择框，支持全部班级筛选  
- **学生姓名**：输入框，支持模糊搜索学生姓名

### 3. 数据列表字段
表格包含以下字段：
- **班级**：显示班级名称
- **课程**：显示课程名称
- **课程类型**：显示课程类型（直播课、录播课、线下课），使用不同颜色的标签区分
- **时间**：显示课程时间
- **学生总数**：显示参与学生总数
- **提问响应率**：显示提问响应率百分比，根据数值使用不同颜色显示
- **学生提问**：显示提问数量，点击可查看详细提问列表
- **学生演示**：显示演示次数，点击可查看详细演示列表
- **情绪捕捉**：显示情绪状态，点击可查看详细情绪分析

### 4. 交互功能

#### 学生提问详情弹窗
- 点击"学生提问"数字可查看详细提问记录
- 显示学生姓名、问题内容和提问时间
- 支持滚动查看多条记录

#### 学生演示详情弹窗
- 点击"学生演示"数字可查看详细演示记录
- 显示学生姓名、演示主题和演示时间
- 支持滚动查看多条记录

#### 情绪捕捉详情弹窗
- 点击"情绪捕捉"可查看详细情绪分析
- 显示整体情绪状态
- 如有沉闷或消极学生，显示具体学生名单
- 使用不同颜色的标签区分情绪状态

### 5. 数据展示特性
- **分页功能**：支持10/20/50/100条每页显示
- **排序功能**：默认按时间倒序排列
- **筛选功能**：支持多条件组合筛选
- **响应式设计**：适配不同屏幕尺寸

### 6. 样式设计
- 参考出勤页面的设计风格
- 使用Element Plus组件库
- 响应式布局，支持移动端访问
- 清晰的数据层次和视觉引导

## 使用方法
1. 启动开发服务器：`npm run dev`
2. 在浏览器中访问应用
3. 点击顶部导航栏的"数据中心"菜单
4. 选择"教学数据"卡片
5. 点击"课堂互动"标签页
6. 使用筛选条件查看特定数据
7. 点击数字按钮查看详细信息

## 技术实现
- 使用Vue 3 Composition API
- 集成Element Plus UI组件库
- 响应式数据绑定和计算属性
- 模拟数据生成和筛选逻辑
- 弹窗组件和事件处理
