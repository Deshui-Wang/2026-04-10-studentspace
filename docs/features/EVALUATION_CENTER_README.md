# 考评中心页面

## 功能概述

考评中心页面是教师教学档案袋系统的重要组成部分，为教师提供全面的专业发展评估工具。

## 主要功能

### 1. 顶部统计信息
- 显示全校教师参与考评的统计数据
- 包含三个关键指标：
  - 全校100%的教师正在积极参加考评
  - 超过76.9%的教师正在申请职称评选
  - 专业技能提升,职业规划策略工具

### 2. 搜索和过滤功能
- **搜索框**：支持按考评工具名称、描述或标签进行搜索
- **分类过滤器**：
  - 全部
  - 通用考评工具
  - 定向职称评选工具
  - 智能胜任力考核工具
  - 学科技能考核工具
  - 其他考核工具

### 3. 考评工具卡片
每个考评工具卡片包含：
- **标题**：考评工具名称
- **分类标签**：显示工具类型（通用考评/智能胜任等）
- **功能标签**：显示具体的考核项目
- **描述**：详细说明考评内容和适用范围
- **参与人数**：显示正在参与考核的教师数量
- **操作按钮**：参与考核或查看进度

### 4. 交互功能
- **卡片悬停效果**：鼠标悬停时卡片会轻微上浮
- **按钮点击反馈**：点击按钮会显示相应的提示信息
- **动画效果**：页面加载时卡片会有淡入动画
- **空状态处理**：当没有搜索结果时显示友好的空状态页面

## 技术实现

### 前端技术栈
- **Vue 3**：使用Composition API
- **响应式设计**：支持桌面端和移动端
- **CSS Grid**：用于卡片布局
- **CSS动画**：提供流畅的用户体验

## 页面路由

- **路径**：`/ai`
- **组件**：`EvaluationCenter.vue`
- **导航**：通过顶部导航栏的"考评中心"菜单访问

## 响应式设计

- **桌面端**：卡片以网格形式排列，最多3列
- **平板端**：卡片调整为2列布局
- **移动端**：卡片单列显示，搜索框和过滤器垂直排列
