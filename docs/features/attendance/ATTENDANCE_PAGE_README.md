# 出勤页面功能说明

## 页面位置
- 路径：`/teaching-resources/attendance`
- 文件：`src/components/teaching-resources/Attendance.vue`

## 功能特性

### 1. 筛选区域
- **课程筛选**：支持选择全部课程或特定课程（高等数学、线性代数）
- **班级筛选**：支持选择全部班级或特定班级（计算机1班、计算机2班、计算机3班、软件1班、软件2班）
- **学生姓名搜索**：支持按学生姓名搜索（在异常人员中查找）
- **时间筛选**：支持日期范围选择，可以筛选特定时间段的出勤数据
- **导出数据**：支持导出当前筛选结果

### 2. 数据列表字段
- **班级**：显示班级名称
- **课程**：显示课程名称
- **日期**：显示出勤日期
- **学生总数**：显示班级总人数（40-45人）
- **异常人数**：显示异常人数（0-5人），可点击查看详情
- **出勤率**：显示出勤率百分比，根据出勤率显示不同颜色

### 3. 异常人员名单
- 点击异常人数可查看详细的异常人员名单
- 显示学生姓名和异常缘由（迟到、早退、请假）
- 异常缘由用不同颜色的标签显示

### 4. 模拟数据
- **班级数量**：5个班级
- **课程数量**：2个课程
- **每班人数**：40-45人
- **异常人数**：0-5人
- **数据时间范围**：2024年全年，每周生成一条数据

### 5. 出勤率颜色标识
- **95%以上**：绿色（优秀）
- **90-94%**：蓝色（良好）
- **80-89%**：黄色（一般）
- **80%以下**：红色（较差）

### 6. 异常缘由标签颜色
- **迟到**：橙色警告标签
- **早退**：红色危险标签
- **请假**：蓝色信息标签

## 技术实现
- 使用Vue 3 Composition API
- 使用Element Plus UI组件库
- 支持响应式设计
- 支持分页功能
- 支持数据筛选和搜索

## 使用方法
1. 访问 `/teaching-resources/attendance` 路径
2. 使用筛选条件过滤数据
3. 点击异常人数查看详细名单
4. 使用分页浏览更多数据
5. 点击导出按钮导出数据
