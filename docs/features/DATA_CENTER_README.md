# 数据中心功能说明

## 功能概述
在顶部导航栏新增了"数据中心"一级菜单，点击后可以访问个人数据空间页面。

## 实现内容

### 1. 导航栏修改
- 在 `src/components/MainNavbar.vue` 中添加了"数据中心"菜单项
- 菜单位置：在"考评中心"和"教学管理"之间
- 路由路径：`/data-center`

### 2. 数据中心页面
- 创建了 `src/views/DataCenter.vue` 页面组件
- 页面设计参考了提供的截图，包含：
  - 页面标题："教师纯享·个人数据空间"
  - 页面描述："每一天都是新起点,智能为您的工作注入无限活力"
  - 五个数据卡片：
    - 教学数据（工作量、班级数、学生数）
    - 资源数据（课程/课件、资产/金额、智能作品）
    - 教学成果（认证、证书、论文）
    - 评价数据（督导、学生、同行好评率）
    - 报告数据（报告总数）

### 3. 路由配置
- 在 `src/router/index.js` 中添加了数据中心页面的路由配置
- 路由路径：`/data-center`
- 组件：`DataCenter`
- 元数据：`{ hideFooter: true }`

## 页面特性

### 视觉设计
- 渐变背景：深蓝色到紫色的渐变
- 毛玻璃效果的卡片设计
- 响应式布局，支持移动端
- 悬停效果和动画过渡

### 数据展示
- 每个卡片包含多个数据项
- 数据项包含数值和标签
- 特殊样式处理（如"好评率"标签）

## 使用方法
1. 启动开发服务器：`npm run dev`
2. 在浏览器中访问应用
3. 点击顶部导航栏的"数据中心"菜单
4. 查看个人数据空间页面
